Developed by Muskan Patel

==============================
Installation Instructions:
==============================

1. Run the installer (LaptopDiagnosticInstaller.exe).
2. Follow the steps to choose components (main app, docs, tools).
3. Optionally create a desktop shortcut.

==============================
Usage:
==============================
- Launch the application from Start Menu or Desktop.
- The tool performs diagnostics on laptop hardware/software.
- Additional tools and documentation are installed in:
  - {app}\tools
  - {app}\docs
